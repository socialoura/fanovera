## 4) Recos d'exécution

- **Match types** : Garder l'exact comme priorité, ajouter le phrase pour la couverture. Pas de broad tant que la conversion n'est pas stabilisée.
- **Enchères** : Démarrer en **Maximiser les conversions sans tCPA** sur 7–10 jours puis activer **tCPA ≈ 5 €** (objectif marge 8,50 €/vente).
- **Whitehat RSA** : conformément à la règle, ne pas mettre "buy", quantités, "pas cher" dans les **titres** d'annonces. C'est OK de **bid sur** "pas cher" en keyword, mais pas dans le texte de l'ad.
- **Suivi** : Re-télécharger le rapport search terms après 7 jours et purger tout nouveau terme à coût > 2 € sans conversion.
